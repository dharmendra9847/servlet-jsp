# 🌍 Dynamic World - Servlet & JSP Web App

A modern, responsive web application built with **Java Servlets**, **JSP**, and **Bootstrap 5**. This project features a sleek Glassmorphism UI, user registration, and a dynamic content dashboard.

## 🚀 Features
- **Modern UI:** Responsive design using Bootstrap 5 and Google Fonts (Inter).
- **Registration & Login:** Clean forms with interactive success modals.
- **Dynamic Dashboard:** A grid-based layout for real-time content display.
- **Form Validation:** Client-side validation and interactive feedback.

## 🛠️ Tech Stack
- **Backend:** Java (Servlets, JSP, JSTL)
- **Frontend:** HTML5, CSS3 (Custom Glassmorphism), JavaScript
- **Styling Framework:** Bootstrap 5
- **Server:** Apache Tomcat 9/10/11
